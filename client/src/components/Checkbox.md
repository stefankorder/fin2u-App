This is the documentation of the app's form checkbox part:

checked

```js
<Checkbox
  value="car"
  field={true}
  userData={{
    car: true,
  }}
  focused={"car"}
  checkboxName="AUTO VORHANDEN?"
/>
```

unchecked

```js
<Checkbox
  value="car"
  field={false}
  userData={{
    car: false,
  }}
  focused={"car"}
  checkboxName="AUTO VORHANDEN?"
/>
```
